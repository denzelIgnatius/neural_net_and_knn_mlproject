1. Please copy the notebook files and the dataset files to the same folder.
2. Open Anaconda Prompt and run the below commands:
	pip install tensorflow==1.15
	pip install keras
3. Open the notebooks using Jupyter notebooks.
4. classify_churn_dataset.ipynb will contain all the executed code for the telecom customer churn dataset.
5. Classify_GPU_dataset.ipynb will contain all the executed code for the GPU dataset.

Note: Please rerun the code for model initialization and running with different training set sizes twice
for each model to see that change in graphes mentioned in the report.
